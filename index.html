2.Described LocalStorage Vs SessionStorage?
LocalStorage:
  • This is storage where data is stored in long time/permanently
  • This storage capability 5/10 MB
  • This stored data pair to pair(key,value) and data type string
  • Manually deleting data from this storage only deletes the data
  • Data cannot be transmitted to the server site and it is supported on most older sites
  • This storage common used method is getItem,setItem,removeItem and clear
SessionStorage:
  • This is storage where data is stored in short time/temporary
  • This storage capability is 5 MB
  • window and tabs closed the data remove in storage
  • This stored data pair to pair(key,value) and data type string
  • Data cannot be transmitted to the server site and it is supported on most older sites
  • This storage common used method is getItem,setItem,removeItem and clear
